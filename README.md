# Crypto Price Telegram Bot 📈

Легкий Telegram-бот на Python для получения актуальных курсов криптовалют. Проект создан для практики работы с внешними API и обработки данных в реальном времени.

## 🚀 Основные функции
* **Запросы к API:** Бот получает данные напрямую с биржевых сервисов (Binance API) через библиотеку `requests`.
* **Интерактивное меню:** Быстрый выбор монет (BTC, ETH, DOGE) с помощью встроенных кнопок (`ReplyKeyboardMarkup`).
* **Безопасность данных:** Все конфиденциальные данные (токены) вынесены в переменные окружения (`.env`).

## 🛠 Технологии
* **Python 3.x**
* **pyTelegramBotAPI (telebot):** простая и надежная библиотека для ботов.
* **Requests:** классика для выполнения HTTP-запросов. 
* **python-dotenv:** стандарт для защиты секретов проекта.
